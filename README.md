# Detecting-abnormal-markets-Early-Warning-Systems
We analyzed a dataset consisting of 43 financial indicators and developed a model to identify early signals of financial crises by detecting initial signs of market instability, thereby helping to minimize financial losses. This enables risk mitigation, strengthening investment strategies and overall improved financial performance.
